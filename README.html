Contributing

Fork this repo and submit a PR to get any link you would like to get included into this repo. Follow these instructions while making contributions:

  1. Add your contribution as a line in the markdown table, make sure to view the rendered README to confirm that the table formatting is not broken.
  2. Make sure that you put an appropriate value in the Category field and useful information in the Description field (the description should not exceed 10 words).
  3. If you are using an existing Category then add your line just after the last line in that Categoryâ€™s table. If you want to add a new category then create a new section for it similar to the other sections.
  4. Prefer adding actionable content such as a code sample or a blog post with code. If you are adding a link to a paper then also include a link to its associated GitHub code repo.
